## Run Locally

Clone the project

```bash
  git clone https://github.com/geekychandan/Angelscoin-crypto-app.git
```

Go to the project directory

```bash
  cd newproject
```

Install dependencies

```bash
  yarn
```

Start the server

```bash
  rarn start
```
